# supplier-wx-ep-ui
taro/react/taro-ui/Ts

# 本仓库为taro 多端小程序 dom
# 技术栈taro-react风格、ts
# ui组件库taro-ui
## 此dom安装依赖后可能会遇到taro-ui引入样式报错需要执行命令
` npm install taro-ui@3.0.0-alpha.3 `




# 新建dom具体步骤
### 1.安装taro脚手架 
####  node 环境（>=12.0.0）
全局安装命令
` npm install -g @tarojs/cli `
### 2.生成初始化项目
` taro init myApp `
### 3.运行项目
` npm run dev:weapp `
` npm run build:weapp `
### 4.具体操作参考
` https://taro-docs.jd.com/taro/docs/GETTING-STARTED `
